# 🏦 Bank Management System (C++)

## 📌 Project Description
This is a console-based Bank Management System developed using C++.  
The application uses file handling to store customer data permanently and provides basic banking operations.

## 🚀 Features
- Create Account
- Display All Accounts
- Search Account by Account Number
- Deposit Money
- Withdraw Money
- Persistent data storage using binary file

## 🛠️ Technologies Used
- C++
- File Handling (Binary Files)
- Object-Oriented Programming (OOP)

## ▶️ How to Run
1. Open the code in any C++ compiler (Cxxdroid recommended)
2. Compile and run the program
3. Use menu options to perform banking operations

## 📂 File Used
- `bank.dat` → stores all account details
